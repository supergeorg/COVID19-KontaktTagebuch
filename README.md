# COVID19-KontaktTagebuch

Die Aufgabe dieser App ist es, die Führung eines Kontakttagebuchs zu vereinfachen. Dies wurde von Prof. Drosten für den Herbst empfohlen, um im Falle einer Infektion alle möglichen Kontaktpersonen finden und informieren zu können.

Die App ist in XCode 12 Beta für iOS 14 entwickelt. Daher wird für ein Ausführen auch diese Vorabversion benötigt.
Bis jetzt wurde KEINE externe Abhängigkeit eingebaut, was die Einfachheit des Quelltexts und Weiterverwendung der App verbessern sollte.
